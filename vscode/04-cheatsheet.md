# VS Code Cheatsheet

## Most Used

| Need | Shortcut |
| --- | --- |
| Command palette | `Cmd+Shift+P` |
| Open file | `Cmd+P` |
| Search project | `Cmd+Shift+F` |
| Toggle terminal | Ctrl+backtick |
| Quick fix | `Cmd+.` |
| Rename symbol | `F2` |
| Format document | `Shift+Option+F` |

## Productivity Tips

- Use profiles for different stacks.
- Keep workspace settings limited to project needs.
- Use tasks for commands the project runs often.
- Use snippets for repeated documentation or code shapes.
- Use source control panel for review, but terminal for exact Git control.

## Interview Productivity Notes

| Situation | Habit |
| --- | --- |
| Live coding | Keep terminal visible |
| Debugging | Use breakpoints instead of only logs |
| Explaining code | Use symbol rename/go-to-definition confidently |
| Time pressure | Search files with `Cmd+P` and project text with `Cmd+Shift+F` |
