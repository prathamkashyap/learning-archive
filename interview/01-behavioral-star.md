# Behavioral and STAR

## Context

Behavioral answers should sound specific. I need real situations, not generic claims like "I am hardworking."

## STAR Format

| Step | What to say |
| --- | --- |
| Situation | Context in one or two lines |
| Task | What responsibility or problem was mine |
| Action | What I actually did |
| Result | What changed, measured if possible |

## Story Bank

Prepare one story for each:

- Conflict or disagreement.
- Tight deadline.
- Bug or failure.
- Leadership or ownership.
- Learning something quickly.
- Teamwork.
- Feedback received.

## Common HR Questions

| Question | What to include |
| --- | --- |
| Tell me about yourself | Education, skills, current focus, projects |
| Strengths | Evidence from a project |
| Weakness | Real improvement area plus action |
| Why this company? | Product, role, skills match |
| Difficult situation | STAR answer |

## Common Mistakes

- Giving opinions without examples.
- Making the story too long.
- Blaming teammates.
- Forgetting the result.

## Quick Reference

```text
Situation -> Task -> Action -> Result
```

Keep answers around 60-90 seconds unless asked for detail.
